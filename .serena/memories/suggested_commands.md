# Suggested Commands

## Build & Run
- `cargo build --release` — release binary → `target/release/lc`
- `cargo run` — dev run
- `./target/release/lc` — direct run

## Test & Lint
- `cargo test` — run tests
- `cargo clippy` — lint
- `cargo fmt` — format
- `cargo fmt --check` — check formatting

## Dev
- `cargo check` — fast compile check

## Git
- `git status`, `git diff`, `git log --oneline -20`

## System (Darwin/macOS)
- `ls`, `find`, `grep`, `rg` (ripgrep)
